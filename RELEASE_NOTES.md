retflo v1.0.0 — initial public release.

## What is retflo

A structured argumentation protocol — a graph of interconnected argument nodes designed to counter capitalist, authoritarian, and status-quo narratives with sourced, composable counterarguments. Built for AI agents, researchers, educators, organizers, debaters, journalists, and anyone doing power analysis.

## Highlights

- **Structured objection handling** — each node maps a common objection to a sourced, citation-backed response with calibrated delivery guidance
- **Agent-native** — installable as a Claude Code skill (`npx skills add retflo/retflo`), with a routing table that maps objection patterns to nodes for autonomous use
- **Graph architecture** — nodes connect across domains with typed, weighted, directional relationships, enabling traversal from any entry point
- **Three access layers** — human-readable HTML, LLM-optimized Markdown, and JSON API, all at [retflo.org](https://retflo.org)
- **Interactive visualizer** — explore the full argument graph visually at [retflo.org/visualizer](https://retflo.org/visualizer/)
- **Cross-domain coverage** — economics, authority, history, philosophy, rhetoric, social structures, and technology
- **Open contribution model** — RCCL v1.0 license (free for cooperatives, democratic orgs, and individuals forever), with CLA for contributors
